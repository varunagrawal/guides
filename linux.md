# Linux Stuff

### Image Modification

#### Compress

```sh
convert test.jpg -quality 50% test-50p.jpg
```

#### Resize

```sh
convert image.jpg -resize 600x400 image.jpg
```

```sh
convert -resize 50% myfigure.png myfigure.jpg
```
